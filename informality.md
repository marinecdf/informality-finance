Subsetting the Data:
====================

I use a for loop to add a "formality" column with constant value 0 to all 10 datasets:

``` r
filenames <- names(which(sapply(.GlobalEnv, is.data.frame)))
for(i in seq_along(filenames)) {
  x <- get(filenames[i])
  for(j in 1:length(x))
  {
    x$formality[[j]] <- 0
  }
  assign(filenames[i], x)
}
```
